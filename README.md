# Python Intro Talk for CS People

This talk was created to give an introduction to python for the U of M Duluth ACM Club.

## Viewing

You can simply open the ```index.html``` file in your preferred browser.

## Editing

To make changes to the content just edit the text in ```index.html```.

To change the style of the presentation you will need to edit the ```python_talk.scss``` file. DO NOT edit the ```python_talk.css``` file it is generated. Run ```make``` to generate ```python_talk.css``` from ```python_talk.scss``` or run the ```make watch``` for live updates.
