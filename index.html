Python vs Java and C++

  • Dynamicly Typed 
    # Python
i = 1
s = "Hello"
    // Java
Integer i = 1;
String s = "Hello";
    // C++
int i = 1;
string s = "Hello";
  • Control Statements 
    # Python
if a == 2 and b > 8:
  print("This is if")
elif b < 8:
  print("This is elif")
else:
  print("This is else")
    // Java
if (a == 2 && b > 8) {
  System.out.println("This is if");
} else if (b < 8) {
  System.out.println("This is else if");
} else {
  System.out.println("This is else");
}
    // C++
if (a == 2 && b > 8) {
  cout << "This is if\n";
} else if (b < 8) {
  cout << "This is else if\n";
} else {
  cout << "This is else\n";
}

Python vs Java and C++

  • Loops 
    # Python
names = ["Alice", "Bob", "Eve"]
for name in names:
  print("It's me:", name)
    // Java
List names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
names.add("Eve");
for (String name : names) {
  System.out.println("It's me: " + name);
}
    // C++
vector names = {"Alice", "Bob", "Eve"};
for (i=names.begin(); i != names.end(); ++i) {
  cout << "It's me: " << *i << endl;
}
  • Functions 
    # Python
def add(a, b):
  return a + b
    // Java
private Integer add(Integer a, Integer b) {
  return a + b;
}
    // C++
int add(int a, int b) {
  return a + b;
}

What other cool data types does Python come with?

  • list
    l = ['hello', "world", 1, 2, 3]
print l[0]
print l[-1]
print l[1:3]
print l[::2]
print l[::-1]
  • set
    s1 = set([1, 2, 3])
s2 = set([3, 4, 5])
print s1.intersection(s2)  # or s1 & s2
print s1.union(s2)  # or s1 | s2
print s1.difference(s2) # or s1 - s2
  • dict
    d = {"a": 1, "b": 2}
print d["a"]
print d["b"]
print d.keys()

What about Classes?

  • Classes
    #!/usr/bin/python

class Person:
  INTRO = "Hi, my name is"

  def __init__(self, name):
    self.name = name

  def say_hello(self):
    print(INTRO, self.name)


alice = Person("Alice")
bob = Person("Bob")
alice.say_hello()
bob.say_hello()
